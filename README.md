# Fake_news_detect

This project is a machine learning-based Fake News Detection system built using Python. It classifies news articles as REAL or FAKE based on their content.

🚀 Features
Classifies news into REAL or FAKE
Uses TF-IDF Vectorization for text processing
Implements machine learning using scikit-learn
Trained on real-world dataset from Kaggle
Simple and easy-to-understand code
📂 Dataset

This project uses the Fake and Real News Dataset from Kaggle:

Fake.csv
True.csv
🛠️ Technologies Used
Python
pandas
NumPy
